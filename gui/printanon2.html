Print Anonymization


1. First generate a printing mask for the specific printer you want to use. Go to: Generate Print Mask

2. Save your document to be anonymized as a PostScript document (*.ps), e.g. using pdf2ps under Linux. Some programs also offer an option for "export as" or "save as" as *.ps file.

3. Load the document in PostScript format (step 2) and the generated printer mask (step 1) and create an anonymous document. This document is saved as "documentname_masked.ps" in the same folder as the document. Additional the settings file is saved as "calibration_masked.txt". Print this file borderless and proceed to step 4.
Generate Document

4. Scan artifacts can cause a small offset between the original and masked yellow dots. To fix this, scan the anonymized print and check the yellow dot patterns. If there is an offset - vary x and y to overlay the masking points exactly with the original yellow points. Repeat step 3 with x and y and click Generate Document again. The dot radius describes the size of a tracking dot to be printed.
